CHESS



1. Whites will start the game and make their first move.
2. Each player will have 60 seconds to make a move.
If a player doesn't make their move within that time, the opponent will be declared the winner.
3. The whole game should be completed within 45 minutes.
After this time, the game will be declared a draw.
4. The game can be paused at any point, but the game board will not be displayed until the game is resumed.
